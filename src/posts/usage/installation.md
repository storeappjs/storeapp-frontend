title: 'Installation'
---

you can install with node package manager (like npm or yarn...)

## ⚡ storeappjs

`storeappjs` is main client.

via (npm)

```bash
$ npm install -g storeappjs
```

via (yarn)

```bash
$ yarn global add storeappjs
```
